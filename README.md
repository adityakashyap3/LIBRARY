# Library Management System 📚

This is a mini project developed using Python.

## Features:
- Add books
- Show books
- Issue books
- Name of student who issued the book
- Number of alloted days
- Date of issuing book
- If book will kept by student more than alloted days then fine will imposed
- Return books

## Technologies Used:
- Python

## How to Run:
1. Open file in Python
2. Run the program
